# 🍽️ Cognifyz Data Analysis Internship

## 📌 Overview
This repository contains all the tasks completed during my **Data Analysis Internship at Cognifyz IT Solution**. The project focuses on data cleaning, exploratory data analysis (EDA), visualization, geospatial analysis, and extracting valuable business insights from restaurant data.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Folium
- Vs Code

---

## 📊 Level 1

### Task 1: Top Cuisines
- Identified the top three most common cuisines.
- Calculated the percentage of restaurants serving each cuisine.
- Visualized cuisine popularity.

### Task 2: City Analysis
- Identified the city with the highest number of restaurants.
- Calculated average ratings by city.
- Determined the city with the highest average rating.

### Task 3: Price Range Distribution
- Analyzed restaurant price ranges.
- Created visualizations for price distribution.
- Calculated the percentage of restaurants in each category.

### Task 4: Online Delivery Analysis
- Determined the percentage of restaurants offering online delivery.
- Compared ratings of restaurants with and without online delivery.

---

## 📈 Level 2

### Task 1: Restaurant Ratings Analysis
- Analyzed the distribution of aggregate ratings.
- Identified the most common rating range.
- Calculated the average number of votes received.

### Task 2: Cuisine Combination Analysis
- Identified common cuisine combinations.
- Evaluated ratings associated with cuisine combinations.

### Task 3: Geographic Analysis
- Plotted restaurant locations using latitude and longitude.
- Identified restaurant clusters and location patterns.

### Task 4: Restaurant Chains Analysis
- Identified restaurant chains present in the dataset.
- Analyzed chain popularity and ratings.

---

## 🚀 Level 3

### Task 1: Restaurant Reviews Analysis
- Extracted common positive and negative keywords.
- Analyzed review lengths.
- Explored the relationship between review length and ratings.

### Task 2: Votes Analysis
- Identified restaurants with the highest and lowest votes.
- Examined correlation between votes and ratings.

### Task 3: Price Range vs Online Delivery & Table Booking
- Analyzed relationships between pricing and service availability.
- Investigated whether higher-priced restaurants provide more services.
